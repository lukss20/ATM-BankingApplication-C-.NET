# ATM / საბანკო აპლიკაცია

ეს არის C# კონსოლის აპლიკაცია, რომელიც არის ბანკომატის სიმულაცია. 

## სატესტო მომხმარებელი

შესასვლელად გამოიყენეთ ეს სატესტო ბარათი:

- ბარათის ნომერი: `1234567812345678`
- ვადის გასვლის თარიღი: `12/30`
- CVC კოდი: `123`
- PIN კოდი: `1234`

## რას აკეთებს აპლიკაცია

აპლიკაცია მომხმარებელს სთხოვს ბარათის ნომერს, ვადის გასვლის თარიღს, CVC კოდს და PIN კოდს. ჯერ მოწმდება ბარათის ნომერი, ვადის გასვლის თარიღი და CVC. თუ ეს მონაცემები სწორია, შემდეგ მოწმდება PIN კოდი. წარმატებული შესვლის შემდეგ იხსნება ბანკომატის მთავარი მენიუ.

მენიუს პარამეტრები:

1. ბალანსის შემოწმება
2. თანხის გამოტანა
3. ბოლო 5 ტრანზაქციის ნახვა
4. თანხის შეტანა
5. PIN კოდის შეცვლა
6. ვალუტის კონვერტაცია
7. გასვლა / საწყის გვერდზე დაბრუნება

## JSON მონაცემების შენახვა

მომხმარებლების მონაცემები ინახება ფაილში:

```text
Data/users.json
```

JSON ფაილი შეიცავს:

- სახელს და გვარს
- ბარათის მონაცემებს: ბარათის ნომერი, ვადის გასვლის თარიღი და CVC
- PIN კოდს
- ბალანსებს GEL, USD და EUR ვალუტებში
- ტრანზაქციების ისტორიას

აპლიკაციის გაშვებისას მომხმარებლების მონაცემები იტვირთება JSON ფაილიდან. ყოველი წარმატებული თანხის შეტანის, თანხის გამოტანის, PIN კოდის შეცვლის ან ვალუტის კონვერტაციის შემდეგ განახლებული მონაცემები ისევ ინახება JSON ფაილში.

თუ JSON ფაილი არ არსებობს, აპლიკაცია ავტომატურად ქმნის სატესტო მომხმარებელს.

## მთავარი კლასები

- `Program.cs` მართავს კონსოლის ეკრანებს, შესვლის პროცესს და მენიუს მუშაობას.
- `User.cs` ინახავს მომხმარებლის პირად მონაცემებს, ბარათის მონაცემებს, ბალანსებს და ტრანზაქციებს.
- `CardDetails.cs` ინახავს ბარათის ნომერს, ვადის გასვლის თარიღს და CVC კოდს.
- `Balance.cs` ინახავს GEL, USD და EUR ბალანსებს.
- `Transaction.cs` ინახავს თარიღს, ტრანზაქციის ტიპს, თანხას, ვალუტას და აღწერას.
- `BankService.cs` ამუშავებს თანხის შეტანას, თანხის გამოტანას, PIN კოდის შეცვლას, ბალანსებს და ვალუტის კონვერტაციას.
- `JsonStorageService.cs` ტვირთავს და ინახავს მომხმარებლებს JSON ფორმატში.
- `LoggerService.cs` მნიშვნელოვან მოვლენებს და შეცდომებს წერს `logs.txt` ფაილში.
- `ValidationService.cs` ამოწმებს ბარათის ნომერს, ვადის გასვლის თარიღს, CVC კოდს, PIN კოდს, ვალუტას და შეყვანილი მონაცემების ფორმატს.

## ვალუტის კონვერტაცია

პროექტში გამოყენებულია ფიქსირებული გაცვლითი კურსები:

- `1 USD = 2.70 GEL`
- `1 EUR = 2.95 GEL`

მომხმარებელი ირჩევს საწყის ვალუტას, სამიზნე ვალუტას და თანხას. აპლიკაცია ამოწმებს ბალანსს, აკლებს თანხას საწყისი ვალუტიდან, ამატებს კონვერტირებულ თანხას სამიზნე ვალუტაში, ინახავს ტრანზაქციას და აახლებს JSON ფაილს.

## ლოგირება

მნიშვნელოვანი მოვლენები იწერება ფაილში:

```text
logs.txt
```

ლოგში ინახება აპლიკაციის გაშვება, წარმატებული შესვლა, წარუმატებელი ვალიდაცია, თანხის შეტანა, თანხის გამოტანა, PIN კოდის შეცვლა, ვალუტის კონვერტაცია და დამუშავებული შეცდომები.
